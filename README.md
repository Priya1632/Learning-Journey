# Learning Journey

A biostatistician's applied learning journal — SAS to R, and into HEOR/RWE.

**Live site:** https://Priya1632.github.io/Learning-Journey/

## Adding a new post

1. Copy `posts/_template.qmd` into a new folder under `posts/`
   - Example: `posts/propensity-score-matching/index.qmd`
2. Fill in the title, date, categories, and content
3. In GitHub Desktop: commit and push
4. Site updates in ~2 minutes

## Folder structure

```
posts/
  ispor-2026/
    index.qmd        ← one folder per post
  your-next-post/
    index.qmd
  _template.qmd      ← copy this for every new post
```

That's it. No other files need to be touched.
